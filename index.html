JohariBazarKinGs Tools

A private internal tool for managing and analyzing the JohariBazarKinGs Etsy shop.

About This Application

JohariBazarKinGs Tools is a personal, internal application built by the owner of the JohariBazarKinGs Etsy shop. It is designed to help streamline day-to-day shop management tasks such as reviewing listings, preparing draft listings, and analyzing shop performance data.

This application is not publicly available and is used exclusively by the shop owner for internal shop operations. It is not a commercial product, and no other sellers or third parties have access to it.

How It Uses the Etsy API

The application connects to the Etsy Open API v3 to perform the following operations for the JohariBazarKinGs shop only:

  • Read shop and listing data — to review current listings, inventory, and sales information in a structured format.
  • Create draft listings — to prepare new product listings as drafts, which are then manually reviewed and published by the shop owner through Etsy's platform.
  • Read sales data — to understand shop performance, track trends, and make informed decisions about inventory and pricing.

Important: This tool does not replace or replicate Etsy's core functionality. All draft listings are reviewed and published manually by the shop owner through Etsy's standard interface. The tool simply assists in organizing and preparing data more efficiently.

Data Handling & Privacy

This application follows strict data handling practices in full compliance with the Etsy API Terms of Use:

  • All data retrieved through the Etsy API is used internally only by the shop owner.
  • No Etsy data is sold, shared, transferred, or exposed to any third-party services, platforms, or individuals.
  • No Etsy member or buyer data is stored beyond what is necessary for internal shop management.
  • The application does not collect, process, or transmit any personal data from Etsy buyers.
  • All data is stored securely on the shop owner's private, password-protected local system.

Technical Details

  • Type: Seller Tool (internal use)
  • Users: Shop owner only (single user)
  • Commercial: No — this is a free, personal tool
  • Data storage: Local / private system only
  • Third-party sharing: None
